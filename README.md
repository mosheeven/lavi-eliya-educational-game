# 砖拽    

Educational game for Lavia and Elia Even featuring three interactive learning modes in Hebrew.

## Features

- **爪  (Sorting Mode)**: Drag-and-drop categorization game
- **爪 驻爪抓 (Popping Mode)**: Timed balloon clicking game
- **爪  (Quiz Mode)**: Multiple-choice quiz with emoji answers

## Topics Covered

-  专 (Dinosaurs)
-   (Space)
-  住驻专 (Numbers)
-  转 (English Letters)

## How to Run

Simply open `single-file-game.html` in any modern web browser. No installation or build process required!

## Technologies

- Konva.js for canvas rendering and animations
- Tailwind CSS for styling
- Web Audio API for sound effects
- Web Speech API for Hebrew text-to-speech

## Language

All content is in Hebrew (注专转) with RTL layout support.
